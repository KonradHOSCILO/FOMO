# FOMO – aplikacja Todo

## Cel projektu
Celem projektu było stworzenie webowej aplikacji typu Todo,
umożliwiającej użytkownikom zarządzanie zadaniami.

Aplikacja została uruchomiona na maszynie wirtualnej
z systemem Linux i napisana w frameworku Django (Python).

Projekt obejmuje:
- rejestrację i logowanie użytkowników (sesje cookies),
- zarządzanie zadaniami,
- obsługę załączników (OVH Object Storage),
- backend oparty o Django i bazę danych.

## Technologie
- Python 3.x
- Django 5.x
- SQLite / MySQL
- OVH Object Storage (S3)
- Linux (VM)
